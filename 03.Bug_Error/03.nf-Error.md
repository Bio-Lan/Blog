# Error

# 1.nextflow run error：

报错信息：  
> `java.lang.InternalError: a fault occurred in a recent unsafe memory access operation in compiled Java code`

原因:  
通常与缺少临时磁盘存储有关，或 nextflow运行时，文件读入内存过程中出现问题


脚本添加：
```
NXF_OPTS="-Dleveldb.mmap=false"
```
